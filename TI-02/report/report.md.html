**Raízes Primitivas** **INE5429 - Segurança em Computadores** Trabalho Individual #2 *Quênio César Machado dos Santos (14100868)* Florianópolis, 10/05/2016 # Raízes Primitivas Um número inteiro *g* é raiz primitiva de um número primo *n* se para todo número inteiro *a* co-primo de *n*: $g^k \equiv a\pmod{n}, k \ge 1$ Raizes primitivas são usadas pelo protocolo de *Diffie–Hellman* para estabelecer uma chave secreta entre duas partes.
